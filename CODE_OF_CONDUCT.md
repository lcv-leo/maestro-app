# Code of Conduct

This project follows the Contributor Covenant Code of Conduct, version 2.1.

## Expected Behavior

- Be respectful, precise, and evidence-oriented.
- Assume good intent while still requiring verifiable claims.
- Protect private editorial material, credentials, and user data.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private drafts, protocols, secrets, logs, or transcripts.
- Submitting fabricated evidence, fabricated citations, or secret-shaped test data.

## Enforcement

The maintainer may remove content, close issues, block contributors, or report abuse when necessary to protect the project and its users.

For security-sensitive matters, use the private reporting path in `SECURITY.md`.
